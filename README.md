## Getting started

1. Rename the file `.env.example` to `.env`
2. Open a new terminal from the VS Code menu - `Terminal > New Terminal`
3. In your new terminal, type the following and hit return:

```sh
npm i
```

```sh

```

5. Finally, from the terminal again, type:

